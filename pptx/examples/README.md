# PPTX Skill Examples

Practical examples for PowerPoint creation and editing.

## Contents

| Example | Description |
|---------|-------------|
| `simple_presentation.py` | Create a basic presentation with python-pptx |
| `extract_text.sh` | Extract text content using markitdown |
| `analyze_structure.py` | Analyze slide layouts and placeholders |

## Quick Start

```bash
# Create a simple presentation
python simple_presentation.py

# Extract text from existing presentation
./extract_text.sh presentation.pptx

# Analyze presentation structure
python analyze_structure.py presentation.pptx
```

## Notes

- For KPMG-branded presentations, always use the template-based workflow in SKILL.md
- These examples are for general PowerPoint manipulation
- See `scripts/kpmg_reference.py` for KPMG-specific implementations
