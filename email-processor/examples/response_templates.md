# Email Response Templates

## Meeting Acknowledgment

```
Hi {{sender_first_name}},

Thanks for reaching out. I'd be happy to meet.

Could you share a few time options that work for you? Alternatively, feel free to book directly: [calendar link]

Best,
{{my_name}}
```

## Request for More Info

```
Hi {{sender_first_name}},

Thanks for your email. To help you better, could you clarify:
- {{question_1}}
- {{question_2}}

Once I have this, I'll get back to you promptly.

Best,
{{my_name}}
```

## Acknowledging Receipt

```
Hi {{sender_first_name}},

Got it — thanks for sending this over.

I'll review and get back to you by {{target_date}}.

Best,
{{my_name}}
```

## Delegating / Forwarding

```
Hi {{sender_first_name}},

Thanks for reaching out. I've looped in {{colleague_name}} who handles this area — they'll be able to help.

Best,
{{my_name}}
```

## Polite Decline

```
Hi {{sender_first_name}},

Thanks for thinking of me. Unfortunately, I'm not able to take this on right now due to current commitments.

{{optional: alternative suggestion}}

Best,
{{my_name}}
```

## Out of Office Auto-Reply

```
Hi,

Thanks for your email. I'm currently out of office until {{return_date}} with limited access to email.

For urgent matters, please contact {{backup_contact}}.

I'll respond to your message when I return.

Best,
{{my_name}}
```
