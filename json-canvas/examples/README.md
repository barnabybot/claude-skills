# JSON Canvas Examples

Sample `.canvas` files demonstrating common patterns.

## Files

| Example | Description |
|---------|-------------|
| `project-map.canvas` | Project planning with goals, tasks, and linked notes |
| `meeting-flowchart.canvas` | Decision flowchart with colored nodes and labeled edges |

## Node Colors

Canvas supports colors 1-6:
- `1` - Red
- `2` - Orange  
- `3` - Yellow
- `4` - Green
- `5` - Cyan
- `6` - Purple

## Usage

Copy these files to your Obsidian vault and open to see them rendered.
Or use as templates when generating new canvases.
